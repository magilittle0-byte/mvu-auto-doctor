# 更新日志

## v1.7.2（2026-07-24）

### 新增
- **通知级别设置**：设置页新增「通知级别」下拉（全部 / 只弹警告与失败 / 静默），成功类 toast 不再强制打扰；所有状态仍完整记入操作时间线。
- **最近操作时间线**：悬浮面板「工具」页新增可折叠的操作时间线，保留最近 30 条带时间戳的状态记录（变量 / 硬合同 / 世界 / 论坛），历史状态不再被最新一句覆盖。
- **工程基建**：新增 `package.json`（`npm test` 一键回归）、MIT `LICENSE`、GitHub Actions CI（push/PR 自动跑全部测试）、版本一致性测试（manifest / index.js / package.json 三处版本必须一致）。

### 修复与增强
- **悬浮球状态光效改为 kind 聚合**：不再对状态文案做正则匹配，改用各状态显式传入的 kind；今后调整文案不会再让球体光效失灵（潜在 bug 修复）。
- **零改动修正版拦截**：模型偶尔原样返回正文却声称已修正，现在会在本地守卫阶段直接拒绝，不再生成无意义的新 swipe。
- **变量诊断"成稿纪律"提示**：system 提示新增一条指令，要求推理模型内部一遍推理完成后一次性输出全部区块，减少 32K 预算被反复重想耗尽导致的截断。

### 版本
- manifest.json / index.js / package.json 统一升级至 1.7.2。
